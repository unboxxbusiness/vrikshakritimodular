# **App Name**: Vrikshakriti

## Core Features:

- Responsive Navigation: A top navigation bar that adapts to different screen sizes.
- Customizable Hero Section: A hero section with editable title and description, offering visual impact.
- Grid Content Display: A grid layout for showcasing featured content cards.
- Consistent Footer: A footer section with contact information and copyright notice.

## Style Guidelines:

- Primary color: Muted blue (#6699CC) to evoke a sense of calm and professionalism.
- Background color: Soft gray (#F0F0F0), providing a light, clean backdrop.
- Accent color: Light blue (#ADD8E6) for highlighting key interactive elements and call-to-actions.
- Body and headline font: 'Inter' sans-serif, chosen for its modern, machined, neutral aesthetic and readability across headlines and body text.
- Simple, line-style icons that complement the minimalist design.
- Use a responsive layout to adapt to different screen sizes.
- Subtle transitions and animations for a smooth user experience.
